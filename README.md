# Civil Premium Demo

Base limpia de una landing comercial one-page para ingeniería civil.

## Stack
- Next.js 16
- TypeScript
- Tailwind CSS 4

## Scripts
```bash
npm run dev
npm run build
npm run start
npm run lint
```
