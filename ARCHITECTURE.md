# Civil Premium Demo — Architecture

## Tipo de proyecto
Landing premium comercial de alto impacto para estudio de ingeniería civil.

## Objetivo
Captar clientes reales del rubro y funcionar como pieza bandera visual/comercial de Marin_Dev.

## Principios
- claridad comercial
- credibilidad técnica
- impacto visual controlado
- cero sobreingeniería
- una sola narrativa visual fuerte
- responsive impecable

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel

## Alcance MVP
- one-page landing
- hero premium con recurso visual técnico
- secciones comerciales esenciales
- CTA claros
- footer simple

## No incluye en fase 1
- backend
- CMS
- 3D interactivo complejo
- multipágina
- blog
- portfolio extenso

## Secciones
1. Header
2. Hero
3. Qué resolvemos
4. Servicios
5. Cómo trabajamos
6. Tipos de intervención
7. Bloque de confianza
8. CTA final
9. Footer

## Dirección visual
Ingeniería estructural premium:
- sobria
- técnica
- precisa
- elegante
- no ornamental

## Regla principal
El hero es el gesto visual protagonista.
El resto acompaña.
